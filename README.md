# RoomieSplit - Expense Management System

A Java + Hibernate based Expense Management System for roommates.

## Features
- Add shared expenses
- Split bills equally
- Check who owes whom
- Track savings goals
- View expense history
- Simple Java Swing UI

## Tech Stack
- Java
- Hibernate ORM
- MySQL
- Maven
- Swing UI

## Modules
- User Management
- Expense Tracking
- Balance Calculation
- Savings Goal Tracker

## How to Run
1. Clone the repository
2. Create MySQL database `expense_db`
3. Update `hibernate.cfg.xml`
4. Run `MainApp.java`
